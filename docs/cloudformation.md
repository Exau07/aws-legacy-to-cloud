# CloudFormation – Équivalence Terraform

Ce projet fournit une implémentation Terraform et une équivalence CloudFormation
pour démontrer la compréhension des concepts AWS indépendamment de l'outil.

## Pourquoi CloudFormation
- Outil natif AWS
- Gestion automatique de l'état
- Très présent dans les environnements AWS-first

## Cas d'usage
- Environnements existants
- Stacks hérités
- Organisations très standardisées
