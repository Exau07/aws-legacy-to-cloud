# CloudFormation – Déploiement applicatif

Cette stack CloudFormation est l'équivalent fonctionnel
de l'implémentation Terraform du projet.

## Composants
- Application Load Balancer
- Auto Scaling Group EC2
- Security Groups
- Rôle IAM EC2
- User Data pour le déploiement applicatif

## Objectif
Démontrer la maîtrise des concepts AWS indépendamment de l'outil IaC.
